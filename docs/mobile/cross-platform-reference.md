# Cross-Platform Reference

## Overview
OneSurvey supports desktop, tablet, and phone. Each device is optimized for a different work style.

## Best Fit by Device
| Device | Best for |
| --- | --- |
| Desktop | Full planning, site management, and organization administration. |
| Tablet | High-capability field execution and survey updates. |
| Phone | Fast updates, capture, and review while moving on site. |

## Typical Guidance
- Use desktop for heavy planning and admin workflows.
- Use tablet/phone for fast field updates, photos, assignments, and tickets.
