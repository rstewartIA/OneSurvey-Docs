# Mobile Settings and Notifications

## Overview
Settings lets each user manage profile, password, display preferences, and notifications.

## Settings Areas
- Profile.
- Password.
- Display.
- Notifications.

## Notification Controls
Users can control email and in-app notifications for:
- Assignments.
- Comments and mentions.
- Ticket updates.
- Site/access updates.

Users can also set assignment due-date reminder timing.
