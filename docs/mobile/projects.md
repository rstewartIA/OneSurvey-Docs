# Mobile Sites

## Overview
Mobile Sites gives fast access to assigned site work while on site.

## What You Can Do
- Open your site list.
- Open site details.
- Jump into surveys, assignments, and tickets based on access.

## Typical Mobile Flow
1. Open the site.
2. Open the target survey.
3. Update element details and upload photos.
4. Update related assignments and tickets.
