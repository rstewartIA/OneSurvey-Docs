## Core User Flow
1. Organization admins invite users and assign access.
2. Users accept the invite, complete profile setup, and sign in.
3. Teams run daily work inside Site Detail tabs.
4. Managers and owners review progress, reports, and audit history.

## Access Basics
- `Role` controls organization authority: Owner, Manager, Member.
- `Seat` controls capability in the app: Full, Field, Viewer.
- Menus and actions change based on each user's role and seat.
