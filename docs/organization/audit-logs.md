# Audit Logs

## Overview

Available to: **Owner**

Audit Logs gives owners a searchable history of key changes across sites, users, assignments, and tickets.

## Common Filters

- Search text
- Site
- User
- Record type
- Event type
- Source
- Date range

## What You Can Review

- Who made the change
- What record changed
- When it happened
- Before and after field values

## Common Use Cases

- Confirm who changed a user's role or seat.
- Review assignment or ticket status changes.
- Validate site updates during handoff or QA.
