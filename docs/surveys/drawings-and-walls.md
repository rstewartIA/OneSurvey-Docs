# Drawings and Walls

## Overview
Drawing tools add visual context such as walls, lines, arrows, shapes, and text.

## Steps
1. Open draw mode.
2. Choose a tool.
3. Draw on the canvas.
4. Select items to adjust or remove.

## Wall Sections
Use wall sections to break large areas into clearer, reviewable segments.
