# Upload a Floor Plan

## Overview
Upload a floor plan to create the survey canvas your team will use for element placement, documentation, and tracking.

## Start the Upload
1. Open your site and go to the **Surveys** tab.
2. Select **Add Floor Plan**.
3. Choose a supported file:
   - JPEG/JPG
   - PNG
   - GIF
   - WebP
   - SVG
   - PDF (single page or multi-page)

If you upload a multi-page PDF, select which pages to include. Each selected page becomes its own survey/floor plan.

## Name and Scale Before Upload
1. Enter a survey name for each selected page.
2. Select **Set Scale** (or **Update Scale**).
3. In the scale screen, double-click to draw a reference line on the plan.
4. Enter the real-world length (Feet).
5. Continue, then select **Upload**.

Scale is required and is used for measurement-based features across the survey experience.

## Tips
- Keep floor plan files clean and high-contrast for easier review.
- Use clear names for each floor or area.
- Verify scale first so distance-based results stay accurate.
- If your layout changes, update the floor plan and re-check scale before continuing work.
