# Drag and Drop Elements

## Overview
Use drag and drop to place elements from the library onto the survey canvas.
This is the fastest way to build out camera, device, and field-condition coverage across a floor plan.

## Open the Element Library
- Use **Favorites** for your most-used items.
- Use **Universal Elements** for shared standard elements.
- Use **My Library** for your organization-specific elements.

Favorites are best for repeat workflows and support up to 20 saved elements.

## Place Elements on the Canvas
1. Open a survey.
2. From the left library, select an element.
3. Drag the element onto the exact location on the floor plan.
4. Drop to place it.
5. Confirm the success message, then continue placing or edit details.

When placement succeeds, you will see an **Element added to floorplan** confirmation.

## What Happens After Drop
- The placed element is automatically selected so you can update details right away.
- A default FoV and standard element options are prepared for the new element.
- If you place the same element type multiple times, OneSurvey keeps placement order clear for review.

## Tips
- Place elements at real-world reference points for cleaner reporting and handoff.
- Update key details in [Element Information](element-information.md) after placement.
- Use [FoV Adjustment](fov-adjustment.md) to verify direction, angle, and depth before final review.
- Add evidence quickly with [Link Photos](link-photos.md) and context with [Commenting](commenting.md).
