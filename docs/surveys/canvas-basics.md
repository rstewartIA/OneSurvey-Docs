# Canvas Basics

## Overview
The canvas is the main survey workspace for placing and updating elements.

## Core Actions
- Pan and zoom.
- Select and move elements.
- Open element details.
- Switch floor plans or versions.

  <div class="os-screenshot-card">

  </div>
  <p class="os-screenshot-caption">Set scale to keep measurements accurate before mapping.</p>

## Practical Workflow
1. Open the target floor plan.
2. Place or select elements.
3. Update element details.
4. Use filters to focus your view.
