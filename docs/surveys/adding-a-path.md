# Adding a Path

## Overview
Use Paths to show cable runs, route direction, or connection guidance for an element on the floor plan.

## Open Path Tools
1. Open a survey and select an element on the canvas.
2. In the [Element Sidebar](design.md), open **Path**.
3. Choose a line color before you start.

If you do not see the **Path** option, your current mode or seat may not include path editing.

## Create and Extend a Path
1. Click **Add Line** to create the first path section for the selected element.
2. Click **Add Section** to extend the route one segment at a time.
3. Repeat until the route matches your intended run.

Each added section continues from the end of the previous section, so the route stays connected.

## Edit Path Shape on the Canvas
- Click a path to select it.
- Drag a start or end marker to reposition that point.
- Neighboring sections update with the move so the route remains continuous.
- Length values update automatically as you move points.

## Remove Sections or Delete a Line
- Click **Delete Section** to remove the last section from the selected line.
- If a line has only one section left, deleting that section removes the full line.

## Review Length and Visibility
- The sidebar shows:
  - selected line length
  - total length across lines on the selected element
  - unit in **Feet**
- Use [Visibility Filters](visibility-filters.md) to toggle **Show Path** on or off during review.
