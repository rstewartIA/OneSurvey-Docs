# Element Information

## Overview
Element Information is where you review and update details for each placed element.

## Typical Fields
- Name or label.
- Status.
- Installation details.
- Survey-specific details.
- Linked comments and photos.

## Steps
1. Select an element.
2. Open the element panel.
3. Update required fields.
4. Save changes.
