# Link Photos

## Overview
Link photos to elements to keep visual documentation tied to the exact location.

## Steps
1. Select an element.
2. Open element photos.
3. Upload one or more images.
4. Confirm the images are linked to the correct element.

## Common Actions
- Preview photos.
- Annotate photos.
- Move a photo to another element.
- Remove photos.
