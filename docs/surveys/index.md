# Surveys Overview

## Overview
Surveys are where teams build floor plan layouts and keep site details up to date.

## What You Can Do
- Upload and scale floor plans.
- Place elements.
- Add element details and photos.
- Add comments.
- Use drawing tools for context.
- Save and restore versions.

## Typical Workflow
1. Start a survey from Site Detail.
2. Upload and scale the floor plan.
3. Place elements and update element information.
4. Add photos and comments.
5. Save versions and review changes.
