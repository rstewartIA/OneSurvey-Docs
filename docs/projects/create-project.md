# Create a Site

## Overview
Create a site to set up your team workspace for surveys, media, files, assignments, and tickets.

## Steps
1. Open **Sites**.
2. Select **Add Site**.
3. Enter core details such as site name and site ID.
4. Select the client, if applicable.
5. Save and open the new site.

## What to Do Next
- Add your first survey and floor plan.
- Confirm site elements and site data.
- Start photo capture in Gallery.
- Set up assignments and tickets.
