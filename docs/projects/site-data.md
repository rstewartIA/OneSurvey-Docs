# Site Data

## Overview
Site Data provides table-based views of survey and element information across the site.

## Common Uses
- Review element details at scale.
- Filter by status, type, and other fields.
- Validate installation and survey details.
- Support report-ready checks before handoff.

## Data Areas
- Survey information.
- Element information.
- Installation information.
- Bill of materials context.
