# Sites Overview

## Overview
Sites are the main workspace for planning, field execution, and final handoff.

## What Teams Do in Sites
- Build and update surveys.
- Review site elements and structured site data.
- Store site files.
- Manage photos and albums in Gallery.
- Track work in assignments.
- Track service issues in tickets.
- Export reports.

## Typical Workflow
1. Create or open a site.
2. Build survey layouts and update element details.
3. Capture photos and organize albums.
4. Track active work in assignments and tickets.
5. Generate reports for review and delivery.

## Access Notes
What users can view or edit depends on role and seat.
