# Site Detail

## Overview
Site Detail is the main workspace where teams execute site work.

## Current Tabs
- Surveys
- Site Elements
- Site Data
- Attachments
- Reports
- Gallery
- Tickets
- Assignments

## Typical Workflow
1. Start in **Surveys** for floor plan and element updates.
2. Use **Gallery** for photos and album organization.
3. Use **Assignments** for owner-based execution work.
4. Use **Tickets** for service issues and follow-up.
5. Use **Reports** for exports and review deliverables.

## Access Notes
Visible tabs and edit actions can differ by role and seat.
