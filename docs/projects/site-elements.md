# Site Elements

## Overview
Site Elements is where teams browse available element types used in surveys.

## What You Can Do
- Search element options.
- Browse by category.
- Mark favorites for faster placement.
- Place selected elements in surveys.

## Why It Matters
Using a consistent element library improves reporting quality and keeps teams aligned.
